# HAIMPANG Assets

Sprint 0 preserves only reusable static assets from the previous project.

## Preserved

- `icons/icon-192.png`: legacy app icon PNG. The source file name said 192, but the actual image is 2048 x 2048.
- `icons/icon-512.png`: legacy app icon PNG. The source file name said 512, but the actual image is 2048 x 2048.

## Empty Asset Buckets

- `backgrounds/`: reserved for preserved or newly created background PNGs.
- `ui/`: reserved for polished UI image assets.
- `legacy/`: reserved for old visual assets whose runtime usage is not decided yet.

No previous JavaScript, CSS, board logic, touch logic, or DOM-grid game code is reused.
